### Install:

1. `git clone git@github.com:vankrajnova/python-project-50.git`
2. `make install`
3. `make package-install`


### How to use:

`gendiff [-h] [-f FORMAT] first_file second_file`

##### Supported file formats:

* json
* yaml/yml

##### Supported output formats:

* stylish
* plain
* json
